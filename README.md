# Divyashu - HTML Resume

A clean and lightweight resume template built using plain HTML5.

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

* A **web browser** (Chrome, Firefox, Safari, Edge, etc.)
* A **text editor** (VS Code, Notepad++, Sublime Text) for editing.

### Installation & Setup

1. **Create the file:** Create a new file in your folder named `index.html`.
2. **Add the code:** Paste the provided HTML code into `index.html` and save the file.
3. **Run the project:** Double-click `index.html` to open it in your default web browser.
